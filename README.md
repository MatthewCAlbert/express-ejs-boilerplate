# Express-EJS My Personal Boilerplate

## Using
- ExpressJs
- EJS (added custom extends function)
- Joi as validator
- SASS middleware
- CSRF (csurf) enabled
- JWT (jsonwebtoken) optional
- LowDB (json based DB) optional
- Mocha Testing

## Folder Structure
```
|- public
|- test
|- src
  |- css (sass sources)
  |- db
  |- handlers
  |- middlewares
  |- models
  |- routes
  |- views
    |- components
    |- pages
```
